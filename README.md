# Inleiding
    De WanderWish App,
    deze applicatie is gemaakt voor de organisatie WanderWish. ze waren opzoek naar een app waar mensen een bucketlist konden maken.
    in deze bucketlist kan de gebruiket locaties toevoegen waar hij/zij nog een keer naar toe zou willen.
    je kan deze locaties ook afvinken of je deze wel of niet bezocht heb.

# Technieken
    tijdens het ontikkelen van de app is gebruikt gemaakt van:

    - Npm: https://www.npmjs.com/
    - Expo: https://expo.dev/go
    - Visual Studio Code: https://code.visualstudio.com/
    - React Native: https://reactnative.dev/
    - Typescript: https://www.typescriptlang.org/
    - git: https://git-scm.com/

# Installatie
    om de app te kunnen runnen heb je de bovenstaande programmas/ packages nodig op je computer.
    mocht je dit nog niet hebben kan je de linkjes volgen om deze te installeren.

    als alles is geinstalleerd kan je de repository clonen naar je computer.
    Heb je de repository gecloned kan je een terminal openen in de WanderWishApp map.

    om de app te laten runnen voer je de volgende commands uit:
    - npm install
    - npx run expo( mocht hij niet starten kan je het proberen met - npx run expo --tunnel)
    - om de app te openen in het web kan je op de w toets drukken in de terminal. 

    als je de app op je telefoon wilt runnen moet je de mobile expo app downloaden en kan je de qr code scannen met je telefoon.

# Keuzenbeschrijving
    er is gebruik gemaakt van react native en expo om de app te ontwikkelen.
    door gebruik te maken van react native hoef je de code maar 1x te schrijven en kan je deze op ios, android en op het web runnen.
    er is gekozen voor expo omdat deit het makkelijk maakt om je app op je telefoon te testen.
